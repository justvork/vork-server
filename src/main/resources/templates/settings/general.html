General

Vork Relay host and system-wide defaults.

Vork Relay Host

Host URL for the zero-knowledge relay service (e.g. https://relay.example.com).

Leave blank to use the default public relay: https://relay.vork.sh.
Set this only when you run your own relay host.

Default OOB Authorization Timeout

How long the relay authorization link remains valid for Telegram and interactive out-of-band requests. Background scheduled jobs use their own per-job timeout. Default: 15 minutes.
