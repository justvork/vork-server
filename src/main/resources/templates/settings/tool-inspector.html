Tool Inspector

Explore platform tools, descriptions, and restricted capabilities.

Tools are the fundamental execution paths provided by the Vork platform — each one represents a discrete, callable capability such as running a terminal command, transferring files, or managing scheduled jobs. A tool is only accessible to the AI during a session if the active Agent Template has been explicitly granted access to it via its allowed-tools list. Unassigned tools are invisible to the model and cannot be invoked.

Restricted — this tool enforces runtime authorisation checks before execution.
